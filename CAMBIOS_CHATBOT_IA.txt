CÓMO PROBAR:
=============

1. Asegúrate de tener Node.js instalado
   - Descárgalo desde: https://nodejs.org/

2. Instala las dependencias (primera vez):
   npm install

3. Inicia el servidor:
   npm start
   O: node server.js

4. Abre index.html en tu navegador

5. Haz clic en el chatbot y escribe una pregunta

TECNOLOGÍA USADA:
=================

Backend:
- Node.js + Express (servidor web)
- Axios (cliente HTTP)
- CORS (permitir solicitudes desde navegador)
- dotenv (variables de entorno)

IA:
- Hugging Face API (inferencia remota)
- Modelo: Mistral-7B-Instruct-v0.1
- Sistema: Prompt engineering personalizado

Frontend:
- JavaScript vanilla
- HTML/CSS
- Fetch API

PROBLEMAS COMUNES:
==================

❌ "Cannot find module 'axios'"
   → Ejecuta: npm install

❌ "El chatbot no responde"
   → Verifica que el servidor esté en http://localhost:5000
   → Abre F12 en el navegador y busca errores

❌ "Las respuestas tardan mucho"
   → Normal la primera vez (2-3 segundos)
   → Es la latencia de la API de Hugging Face

❌ "El servidor no inicia"
   → Verifica que el puerto 5000 esté disponible
   → Intenta otro puerto: PORT=3000 node server.js
