📊 ALMACENAMIENTO DE DATOS
════════════════════════════════════════════════════════════════════════════════

Los datos se guardan en localStorage (navegador) en 3 arrays:

localStorage['adoptionSubmissions']
├─ Cada solicitud contiene:
│  ├─ name, email, phone, dog, reason, experience
│  ├─ houseType, yard, timestamp
│  └─ Se añade un nuevo objeto cada vez que envías

localStorage['donations']
├─ Cada donación contiene:
│  ├─ amount (monto), email, recurring (recurrente)
│  ├─ timestamp
│  └─ Se añade un nuevo objeto cada vez que donas

localStorage['reports']
├─ Cada reporte contiene:
│  ├─ name, email, phone, location, breed, size
│  ├─ urgency, description, hasPhoto, timestamp
│  └─ Se añade un nuevo objeto cada vez que reportas

❓ PREGUNTAS FRECUENTES
════════════════════════════════════════════════════════════════════════════════

P: ¿Dónde están guardados los datos?
R: En localStorage del navegador. Presiona F12 → Application → LocalStorage

P: ¿Se envían los datos a un servidor?
R: No, solo se guardan localmente. Para producción, necesitarías backend.

P: ¿Puedo borrar los datos?
R: Sí, desde DevTools: Application → LocalStorage → Delete, o código.

P: ¿Funciona sin internet?
R: Sí, funciona completamente sin conexión.

P: ¿Los datos se pierden si cierro navegador?
R: No, localStorage persiste incluso después de cerrar.

P: ¿Puedo modificar el formulario?
R: Sí, está en index.html. Abre con editor de texto.

P: ¿Cómo agrego más perros?
R: En JavaScript, busca const dogsData = [ y agrega nuevo objeto.

P: ¿Puedo cambiar los colores?
R: Sí, busca :root { --primary-color en CSS.
