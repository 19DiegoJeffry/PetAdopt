### 📄 Archivos Principales
- **index.html** - Página web completa (HTML + CSS + JavaScript integrados)
  - Navbar interactivo
  - Hero section atractivo
  - Galería de 6 perros
  - Sección detallada de cuidados (6 categorías)

## 🚀 Cómo Iniciar

### Opción 1: Solo Frontend (Sin Backend)
```bash
# Simplemente abre index.html en tu navegador
# El chatbot funciona con respuestas predefinidas
```

### Opción 2: Con Backend
```bash
# Instalar dependencias
npm install

# Iniciar servidor
npm start

# Abre index.html en navegador
```

### Scripts de Inicio
- **Windows**: Doble clic en `iniciar.bat`
- **Linux/Mac**: `bash iniciar.sh`
