                            CÓMO USAR LA PÁGINA
================================================================================

OPCIÓN 1: RÁPIDO (30 segundos) - SIN INSTALAR NADA
─────────────────────────────────────────────────
1. Busca el archivo "index.html"
2. Haz doble clic para abrirlo en tu navegador
3. ¡Listo! La página funciona completamente

✨ El chatbot funciona con respuestas inteligentes basadas en palabras clave
✨ Todo es local en tu navegador, sin necesidad de servidor


OPCIÓN 2: CON BACKEND (2-3 minutos) - RECOMENDADO
──────────────────────────────────────────────────
1. Asegúrate de tener Node.js instalado
   → Descarga de: https://nodejs.org/

2. Abre una terminal/command prompt en esta carpeta

3. Instala las dependencias:
   npm install

4. Inicia el servidor:
   npm start

5. Verás algo como:
   🐾 Servidor PetAdopt ejecutándose en http://localhost:5000

6. Abre "index.html" en tu navegador
   → Ahora el chatbot conecta con el servidor backend

================================================================================
                        SOLUCIÓN DE PROBLEMAS
================================================================================

❌ "El chatbot no responde"
   └─ Verifica que npm start esté ejecutándose
   └─ Abre F12 (herramientas del navegador) para ver errores
   └─ El chatbot tiene modo fallback, debería funcionar igual

❌ "npm no reconocido"
   └─ Node.js no está instalado
   └─ Descárgalo: https://nodejs.org/
   └─ Reinicia la terminal después de instalar

❌ "Error en puerto 5000"
   └─ Otro proceso usa ese puerto
   └─ Cambia en server.js: const PORT = 5001;

❌ "No veo index.html"
   └─ Asegúrate de estar en la carpeta correcta
   └─ En Windows: dir (comando)
   └─ En Mac/Linux: ls (comando)
