Opción 2: Con Backend Node.js (2-3 minutos)

Ejecuta el servidor backend para mejor experiencia del chatbot con IA.

Requisitos:

  • Node.js instalado (descarga desde nodejs.org)

Pasos:

1️⃣ Instalar dependencias

npm install

2️⃣ Iniciar servidor

npm start

Verás en la terminal:

🐾 Servidor PetAdopt ejecutándose en http://localhost:5000

3️⃣ Abrir página en navegador

Abre index.html (el chatbot ahora conecta con el servidor)

✅ Ventajas:
• Mejor rendimiento del chatbot
• Escalable para futuras mejoras
• Listo para IA avanzada
⚠️ Nota: Necesitas Node.js instalado. Si no lo tienes, descárgalo antes.
Instrucciones Backend →

❓ Solución de Problemas

¿El chatbot no responde?

  • El backend está en pausa (verifica que npm start esté ejecutándose)
  • Abre las herramientas de desarrollador (F12) para ver errores
  • El chatbot tiene modo fallback y funciona sin servidor

¿npm no reconocido?

  • Node.js no está instalado correctamente
  • Descarga desde: https://nodejs.org/
  • Reinicia tu terminal después de instalar

¿Puerto 5000 en uso?

  • Cambia el puerto en server.js en la línea: const PORT = 5001;

🎨 Personalización Rápida

Todos los cambios se hacen directamente en index.html:

Cambiar nombre del sitio

Busca <title>🐕 PetAdopt</title> y cambia el nombre

Cambiar colores

En el <style>, edita:

--primary-color: #FF6B6B; /* Color principal */
--secondary-color: #4ECDC4; /* Color secundario */

Agregar/Cambiar perros

En el <script>, busca const dogsData y edita la lista

Cambiar información de contacto

Busca la sección de footer y reemplaza los datos
