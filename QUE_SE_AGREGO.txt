1. SECCIÓN "SOLICITAR ADOPCIÓN"
   ════════════════════════════════════════════════════════════════════════
   
   Ubicación en página: Entre sección "Adoptar" y "Donaciones"
   Anchor HTML: #solicitar
   Clase CSS: .solicitar-adopcion
   
   FORMULARIO CON CAMPOS:
   ├─ Nombre Completo (text) - REQUERIDO
   ├─ Correo Electrónico (email) - REQUERIDO
   ├─ Teléfono (tel) - REQUERIDO
   ├─ Seleccionar Perro (select) - REQUERIDO
   │  └─ Opciones: Max, Bella, Charlie, Lucy, Rocky, Daisy
   ├─ ¿Por qué deseas adoptar? (textarea) - REQUERIDO
   ├─ Experiencia previa (textarea) - REQUERIDO
   ├─ Tipo de vivienda (select) - REQUERIDO
   │  └─ Opciones: Casa, Apartamento, Otra
   ├─ ¿Tienes patio? (select) - REQUERIDO
   │  └─ Opciones: Sí, No
   └─ Botón ENVIAR (type: submit)
   
   FUNCIONALIDAD:
   ├─ Validación automática de campos
   ├─ Mensaje de alerta si falta completar
   ├─ Almacenamiento en localStorage → "adoptionSubmissions"
   ├─ Confirmación personalizada: "¡Gracias [nombre]! Tu solicitud..."
   ├─ Limpieza automática del formulario
   └─ Timestamp automático de cada envío
   
   ESTILOS:
   ├─ Fondo gris claro (#f8f9fa)
   ├─ Formulario blanco con sombra
   ├─ Grid responsivo (2 columnas en desktop, 1 en móvil)
   ├─ Colores consistentes con tema
   └─ Validaciones visuales


2. SECCIÓN "DONACIONES"
   ════════════════════════════════════════════════════════════════════════

   Clase CSS: .donaciones
   
   CONTENIDO INFORMATIVO:
   ├─ Título: "Apoya Nuestro Refugio"
   ├─ 3 Tarjetas de impacto:
   │  ├─ 🐕 Rescate - "Rescatamos perros de situaciones de riesgo"
   │  ├─ 🏥 Atención Médica - "Cuidados veterinarios completos"
   │  └─ 🏠 Refugio - "Alojamiento seguro y cómodo"
   
   FORMULARIO DE DONACIÓN:

   ├─ Botones rápidos rellenan automáticamente el campo de cantidad
   ├─ Botón activo se resalta visualmente
   ├─ Validación: monto debe ser > 0
   ├─ Almacenamiento en localStorage → "donations"
   ├─ Confirmación personalizada: "¡Gracias por tu donación de $X!"
   ├─ Indica si es donación recurrente
   ├─ Limpieza automática
   └─ Timestamp automático
   
   DISEÑO:

   ├─ Texto blanco
   ├─ Tarjetas con fondo semi-transparente
   ├─ Formulario blanco con sombra
   ├─ Botones interactivos

CAMBIOS EN JAVASCRIPT
═══════════════════════════════════════════════════════════════════════════════

FUNCIÓN: initializeFormHandlers()
└─ Se ejecuta al cargar la página
└─ Conecta eventos a los 3 formularios

FUNCIÓN: handleAdoptionSubmit(e)
├─ Valida campos del formulario
├─ Guarda en localStorage["adoptionSubmissions"]
├─ Muestra confirmación personalizada
└─ Limpia el formulario

FUNCIÓN: selectDonationAmount(e)
├─ Se ejecuta al hacer clic en botones de monto
├─ Llena automáticamente el campo de cantidad
├─ Resalta el botón activo
└─ Permite cambiar de opción

FUNCIÓN: handleDonationSubmit(e)
├─ Valida cantidad > 0
├─ Guarda en localStorage["donations"]
├─ Muestra confirmación con monto y frecuencia
├─ Limpia formulario y resalta
└─ Remueve clase "active" de botones

FUNCIÓN: previewPhoto(event)
├─ Se ejecuta cuando selecciona foto
├─ Lee el archivo con FileReader
├─ Muestra preview HTML
└─ Mantiene actualizado el DOM

FUNCIÓN: handleReportSubmit(e)
├─ Valida campos requeridos
├─ Crea objeto con datos del reporte
├─ Guarda en localStorage["reports"]
├─ Muestra confirmación con nivel de urgencia
├─ Limpia formulario y preview
└─ Timestamp automático
